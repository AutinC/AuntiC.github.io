进度条

进度条对话框有两种使用方式：

  • 模态方式，简单方便，但是要用**QApplication::processEvents()**使事件循环保持正常进行状态，以确保应用不会被阻塞；
  • 非模态方式，通过QTimer实现定时设置进度条。

Qt提供了两种显示进度条的方式：

  • QProgressBar提供可一种横向或纵向显示进度的控件表示方式，用来描述任务完成情况，如下是QProgressBar的几个重要属性：
    • minimum、maximum：进度条指示的最小值和最大值
    • format：进度条显示文字的格式，有三种：**%p%（显示完成的百分比，默认）、%v（显示当前进度值）、%m**（显示总的步进值）
    • invertedAppearance：使进度条以反方向显示进度
  • QProgressDialog提供针对慢速过程的对话框表示方式，有以下几个属性：
    • minimum、maximum：进度条指示的最小值和最大值
    • minimumDuration：进度条对话框出现前的等待时间，系统算一个完成所有工作需要的时间，如果大于进度条等待时间，就显示进度条对话框，否则不显示，实现无感进度。

调色板与电子钟

QPalette类用于管理对话框的外观显示，相当于对话框或控件的调色板，每个窗体或控件都包含一个QPalette对象，在显示时按照它的QPalette对象中的各个部分各状态下的颜色的描述进行绘制。

QTime类用于获取和显示系统时间。

  • QPalette

    • ColorGroup指三种不同状态下的显示效果

      • QPalette::Active：获得焦点时的状态
      • QPalette::Inactive：未获得焦点时的状态
      • QPalette::Disable：不可用时的状态

      获得焦点和未获得焦点状态在通常情况下颜色显示是一致的，可以设置不同。

    • ColorRole是颜色主题，对窗体不同部位颜色的分类

      • QPalette::Window：背景色
      • QPalette::WindowText：前景色

    • **setColor()**可以设置不同部位、不同状态下的颜色

    • setBrush()通过画刷修改显示，这样就能用图片而不是单一颜色对主题填充

  • 实例

  • QTime

    • **currentTime()**用于获取当前系统时间

    • **toString()**用于将获取的时间转换为字符串类型，需要指定转换后显示格式：

      • H/h：使用H是24小时制，使用h并且加上AM/PM时，使用12小时制，否则仍是24小时制
      • m：分
      • s：秒
      • AP/A：AM/PM
      • Ap/a：am/pm

      也可以直接用Qt::DateFormat作为参数指定显示格式，如Qt::TextDate、Qt::ISODate、Qt::LocaleDate等

#include <QTimer>
#include <QTime>
#include <QMouseEvent>
#include <QDebug>

DigiClock::DigiClock(QWidget* parent): QLCDNumber(parent)
{
    QPalette p = palette();                             // 新建调色板对象
    p.setColor(QPalette::Window, Qt::blue);             // 设置调色板对象为窗体，颜色为蓝色
    this->setPalette(p);                                // 将调色板p应用到实例对象

    this->setWindowFlags(Qt::FramelessWindowHint);      // 设置窗体没有面板边框和标题栏
    this->setWindowOpacity(0.5);                        // 设置窗体透明度（这里为半透明）

    QTimer* timer = new QTimer(this);                               // 新建一个定时器对象
    connect(timer, SIGNAL(timeout()), this, SLOT(showTime()));      // 连接定时器timeout信号与显示时间的槽函数showTime

    timer->start(1000);                                             // 以1000为周期启动定时器
    this->showTime();                                               // 初始化时间显示
    this->resize(240, 60);
    this->showColon = true;
}

void DigiClock::showTime()
{
    QTime time = QTime::currentTime();                        // 获取当前时间并保存在QTime对象中
    QString text = time.toString("hh:mm");                 // 把时间转换为字符串类型


    if(this->showColon)                             // 控制“：”的闪烁
    {
        text[2] = ':';
        this->showColon = false;
    }
    else
    {
        text[2] = ' ';
        this->showColon = true;
    }
    qDebug() << text << endl;

    this->display(text);
}


void DigiClock::mousePressEvent(QMouseEvent *event)
{
    // 如果按下的是鼠标左键，则保存当前鼠标点所在的位置相对于窗体左上角的偏移值dragPosition
    if(Qt::LeftButton == event->button())
    {
        this->dragPosition = event->globalPos() - frameGeometry().topLeft();
        event->accept();
    }

    // 如果按下的是鼠标右键，则退出窗体
    if(Qt::RightButton == event->button())
    {
        this->close();
    }
}

void DigiClock::mouseMoveEvent(QMouseEvent *event)
{
    // 判断按下的有没有鼠标左键
    if(Qt::LeftButton & event->buttons())
    {
        this->move(event->globalPos() - this->dragPosition);
        event->accept();
    }
}
